Hulp krijgen

Als je ooit hulp nodig hebt terwijl je Git gebruikt, hier zijn drie manieren om de gebruiksaanwijzing (manpage) voor elk Git commando te krijgen:

$ git help <actie>
$ git <actie> --help
$ man git-<actie>

Bijvoorbeeld, je kunt de gebruikershandleiding voor het commando git config krijgen door het volgende te typen:

$ git help config

Deze commando’s zijn fijn omdat je ze overal kunt opvragen, zelfs als je offline bent. Als de gebruiksaanwijzing en dit boek niet genoeg zijn en je persoonlijke hulp nodig hebt, kan je de kanalen #git of #github (beiden Engelstalig) op het Freenode IRC netwerk (irc.freenode.net) proberen. Deze kanalen zijn regelmatig gevuld met honderden mensen die allemaal zeer ervaren zijn met Git en vaak bereidwillig om te helpen.
